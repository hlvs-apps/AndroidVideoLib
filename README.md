# AndroidVideoLib
A simple Android Library to do work with Videos based on JCodec.

## Usage:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

and

```
	dependencies {
	        implementation 'com.github.hlvs-apps:AndroidVideoLib:v0.0.2-alpha'
	}
```
