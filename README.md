# AndroidVideoLib
A simple Android Library to do work with Videos based on JCodec
More Information and Javadoc coming soon.

## Usage:

```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```

and

```
	dependencies {
	        implementation 'com.github.hlvs-apps:AndroidVideoLib:v0.0.1-alpha'
	}
```
